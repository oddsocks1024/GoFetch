Bug Reports

The current version of Go Fetch! is still in a beta state which means despite extensive testing, there may still be bugs. If you discover a bug, please E-Mail a bug report containing as much information as you can. Try to include the following.

Machine Description
(Including Model, Processor, RAM, Cards, etc)

AmigaOS Version

TCP/IP Stack you are using
(Including version)

Any Enforcer/Cyberguard/Segtracker/Mungwall output

A good clear desciption of the bug.

If possible, a log file created from Go Fetch!

Details on the FTP site you were using, and the file you were tring to download.

Send bug reports to ian.chapman@bleach.demon.co.uk
