Future

I would like to add the following features to future versions of Go Fetch! If you have any additional suggestions, then please feel free to E-Mail me.

  • Support fot HTTP downloads
  • Resume for HTTP sites that support it.
  • Multiple Simultaneous Downloading
    (Downloading from several sites at the same time)
  • CPS Counter
  • Bandwidth Setting
    (Set a maximum bandwidth Go Fetch! should use)
  • Locale Support
  • Alternative profiles files, chosen on command line
  • Toggle which displays the full log, or minimum info log
  • Keep the connection active if the next file is on the same server
