Requirements

  • MUI 3.7 or higher
  • AmigaOS 3.0 or higher
  • BSD Socket compatible TCP/IP Stack (E.g. Miami, Genesis, Termite etc)
  • Minimum of 512K RAM Free.
  • reqtools.library V38 or higher (Included in distribution)

Testing

Go Fetch! has only been tested with Miami and Miami Deluxe TCP/IP stacks, however it should work with any BSD Socket compatible TCP/IP stack. The following stacks are BSD Socket compatible.

AmiTCP
AS255r2
Genesis
Miami
Miami Deluxe
Termite TCP
