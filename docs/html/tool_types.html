Tool Types

ANONPASS

This specifies the default password you want to use for anonymous logins. This should usually be your E-Mail address. If no password is specified then nobody@nobody.com will be used.

DOWNLOAD

This specifies the default download directory you want to use. If no directory is specified then Ram: will be used as the default. Please note that if the directory path ends with a directory rather than a device name you must add the trailing slash. For example:

Work:download/ (Valid Path)
Work: (Valid Path)
Work:download (Invalid Path)

BEEP

If this tooltype is present then GoFetch! will beep (or play system sample etc) when Go Fetch! has finished transferring all profiles.
